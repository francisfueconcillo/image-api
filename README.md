# image-api
An Image API that uploads to Firebase Storage to be compatible for Flutter mobile apps


## Local Setup
- `pnpm i`
- `pnpm start`

## Deployment

- API deployed in [render.com](https://render.com/)
- PROD API BASE URL: https://image-resizer-9pc9.onrender.com

## API Documentation

- GET /image/{image-id}
- POST /image/{image-id}
